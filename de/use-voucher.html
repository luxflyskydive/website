Geschenkgutscheine

Einlösung Ihres Gutscheins.

Drei Schritte vom Code zum Flug.

So verwenden Sie Ihren Gutschein

01

Gehen Sie zu shop.indoorskydive.lu

Besuchen Sie unsere Buchungsseite und wählen Sie Ihr Flugpaket. Wählen Sie Ihr bevorzugtes Datum und Ihre bevorzugte Uhrzeit. Alle Pakete sind verfügbar: Discovery, Sensation, Performance, Progression und Pro Sessions. Bei Bedarf können Sie auch Kinderpakete oder Event-Gruppenbuchungen buchen.

02

Geben Sie Ihren Gutscheincode

An der Kasse sehen Sie ein Feld „Geschenkgutschein“. Fügen Sie dort Ihren Gutscheincode ein. Der Wert wird automatisch auf Ihre Buchung angewendet. Wenn Ihre Buchung mehr kostet als Ihr Gutschein, zahlen Sie die Differenz per Karte an der Kasse oder an der Rezeption am Tag.

03

Schließen Sie Ihre Buchung ab

Schließen Sie den Bestellvorgang ab und Sie erhalten eine Bestätigungs-E-Mail mit allen Details. Erscheinen Sie am Veranstaltungstag mit Ihrer Bestätigung. Sie sind gebucht, bereit und fliegen gleich los.

Häufige Fragen

Kann ich einen Gutschein auf mehrere Sitzungen aufteilen?

Nein. Jeder Gutscheincode kann nur einmal in einer einzigen Transaktion verwendet werden. Wenn Sie mehrere Gutscheine haben, gilt jeder für eine separate Buchung.

Was passiert, wenn mein Gutschein nicht den vollen Preis abdeckt?

Kein Problem. Den Restbetrag können Sie bei der Buchung an der Kasse per Karte oder am Tag Ihrer Sitzung in bar an der Rezeption bezahlen. Der Gutschein gilt als Gutschrift auf den Gesamtbetrag.

Läuft mein Gutschein ab?

Geschenkgutscheine sind 12 Monate ab Kaufdatum gültig. Wenn Sie Ihren Code vor Ablauf nicht nutzen können, können Sie ihn um weitere drei Monate verlängern. Geben Sie dazu Ihren Code auf der Website ein und befolgen Sie die Schritte zur Verlängerung.

Kann ich meinen Gutschein auf jemand anderen übertragen?

Ja. Gutscheine sind nicht an einen bestimmten Namen gebunden. Wer den Gutscheincode hat, kann ihn nutzen. Sie können es einem Freund oder Familienmitglied schenken und dieser kann es selbst buchen und einlösen.

Was passiert, wenn ich meinen Gutscheincode verliere?

Kontaktieren Sie uns mit einem Kaufbeleg (z. B. Ihrem Originalbeleg oder Ihrer Bestell-E-Mail) und wir können Ihren Code erneut ausstellen. Wir führen Aufzeichnungen über alle Gutscheinverkäufe.

Kann ich eine Rückerstattung meines Gutscheins in bar erhalten?

Nein. Gutscheine sind nicht erstattungsfähig und können nicht gegen Bargeld eingetauscht werden. Sie sind 12 Monate ab Kauf gültig. Wenn Sie den Gutschein selbst nicht nutzen können, können Sie ihn auch jemand anderem schenken – Gutscheine sind nicht an einen Namen gebunden.

Bereit zum Fliegen?

Ihr Gutschein wartet darauf, eingelöst zu werden. Gehen Sie zu shop.indoorskydive.lu und buchen Sie jetzt Ihre Sitzung. Wählen Sie Ihr Paket, geben Sie Ihren Code ein und bereiten Sie sich auf das unglaublichste Erlebnis Ihres Lebens vor.
